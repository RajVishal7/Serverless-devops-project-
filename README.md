\# Serverless DevOps CI/CD Pipeline



\## Overview

This project demonstrates a complete serverless DevOps pipeline using AWS and Terraform.



\## Architecture



GitHub → CodePipeline → CodeBuild → Terraform → AWS Lambda → CloudWatch → SNS



\## Features

\- Automated deployment using CI/CD

\- Infrastructure as Code using Terraform

\- Serverless compute using AWS Lambda

\- Monitoring using CloudWatch

\- Email alerts using SNS



\## Tech Stack

AWS, Terraform, GitHub, CI/CD



\## Outcome

Successfully built an automated DevOps pipeline with real-time monitoring and alerting.
